# Next Persian Fonts
پکیج مجموعه فونت‌های فارسی برای Next.js

## Install
```bash
npm install next-persian-fonts
```

## Use
```javascript
import { vazirMatn } from 'next-persian-fonts/vazirmatn'
```
| You can also use `sahel` and `estedad` font.

```javascript
<body className={sahel.className}>...</body>
```
